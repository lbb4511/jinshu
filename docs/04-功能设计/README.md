## 与模块设计的区别

| 类型         | 范围           | 示例                                                |
| ------------ | -------------- | --------------------------------------------------- |
| **模块设计** | 完整的业务领域 | 报表导入模块（包含Excel解析、数据校验、批量写入等） |
| **功能设计** | 单个功能点     | 报表预览水印功能、大任务自动分片功能                |

## 文档命名规范

```
{功能编号}-{功能名称}-设计.md
```

示例：

- `001-报表预览水印功能设计.md`
- `002-大任务自动分片与公平调度设计.md`
- `003-异常数据自动告警设计.md`
