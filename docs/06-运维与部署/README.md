## 生产环境指标基准

| 指标               | 目标值  | 告警阈值 |
| ------------------ | ------- | -------- |
| API 响应时间 P95   | < 500ms | > 2s     |
| API 成功率         | > 99.9% | < 99%    |
| 数据库连接池使用率 | < 70%   | > 85%    |
| 内存使用率         | < 70%   | > 85%    |
| CPU 使用率         | < 60%   | > 80%    |
| 任务队列积压       | < 100   | > 1000   |
