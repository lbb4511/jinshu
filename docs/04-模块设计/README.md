## 📐 设计要求

所有模块设计必须使用模板：[../02-流程与规范/01-个人+AI开发指南/01.设计文档模板.md](../02-流程与规范/01-个人+AI开发指南/01.设计文档模板.md)

必须包含：

1. **设计目标** - 量化指标（如：100万行导入 ≤ 5分钟）
2. **技术选型对比** - 为什么选A不选B
3. **核心架构图** - 数据流转、依赖关系
4. **数据库设计** - 表结构、索引、分区
5. **异常处理** - 失败场景、重试策略
6. **性能考量** - 瓶颈分析、优化方案
7. **安全考量** - 多租户隔离、权限控制
8. **测试方案** - 边界场景、压测方案
9. **上线与回滚方案**
