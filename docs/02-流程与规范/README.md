## 📊 阶段切换指南

### 什么时候从「个人+AI」→「社区协作」

出现以下信号时，开始启用社区协作流程：

✅ 有3个以上的外部贡献者
✅ 每周有5个以上的Issue/PR
✅ 有人开始问"这个功能为什么这么设计？"
✅ 你一个人Review不过来了

---

## 💡 核心理念

**流程不是为了限制，是为了提速。**

- 人少的时候，怎么快怎么来；人多了，再把流程加上。

不要在只有1个人的时候，就用上100个人的流程。也不要100个人了，还在用1个人的流程。

**合适的才是最好的。**
