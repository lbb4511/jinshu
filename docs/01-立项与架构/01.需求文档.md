## 1. 项目概述

"锦书"系统旨在构建一套高性能、高可用的企业级全生命周期报表管理平台。系统基于云原生架构，支持从多源数据导入、动态报表设计、全生命周期审批（创建-预览-审查-公开），到多格式（动态网页、静态HTML、分段PDF、CMYK印刷级PDF）输出的全流程自动化。

核心目标是解决百页级复杂报表（含大量图表）的生成性能瓶颈，并实现动静页面的组件复用与灵活打印。

## 2. 核心功能需求

### 2.1 报表全生命周期管理

- 支持报表的创建、预览、审查、驳回、公开等全流程状态管理
- 支持报表元数据版本控制，确保历史报表能正确渲染
- 支持基于用户角色的权限控制（制表人、审查人、访客）

### 2.2 动静同构报表展示

- 支持动态网页模式与静态HTML模式的组件复用
- 支持报表实时预览
- 支持打印分页控制，防止内容被拦腰截断

### 2.3 数据导入与ETL

- 支持Excel（.xlsx/.xls）和CSV文件批量导入
- 支持文件上传校验（文件类型、大小限制）
- 支持数据导入三级校验：格式校验、类型校验、业务校验
- 支持导入错误日志记录与错误率监控
- 支持导入进度实时反馈
- 支持外部系统数据同步（可选）

### 2.4 数据导出

- 支持Excel（.xlsx）格式导出（≤100万行）
- 支持CSV格式导出（超大数据量）
- 支持导出前数据量预估（行数、文件大小、推荐格式）
- 支持导出进度实时反馈
- 支持Excel模板驱动模式（可选）

### 2.5 高性能PDF生成

- 支持静态HTML转PDF
- 支持大报表分段渲染与合并
- 支持CMYK印刷级PDF色彩转换
- 支持PDF生成进度实时反馈

## 3. 非功能需求

### 3.1 性能需求

- 单文件最大支持500MB导入
- 支持百万级数据行导出
- 支持百页级复杂报表（含大量图表）生成
- 报表预览响应时间≤3秒

### 3.2 可靠性需求

- 系统高可用，支持容灾备份
- 数据导入失败率阈值控制（>30%自动终止）
- DDL变更自动备份

### 3.3 安全性需求

- 防止文件后缀伪造（Magic Number校验）
- 敏感数据加密存储
- 数据源连接信息安全管理

### 3.4 可维护性需求

- 云原生架构，支持容器化部署
- 支持水平自动扩缩容
- 支持定时任务调度

### 3.5 多租户需求

- 数据严格租户隔离，所有业务数据绑定租户ID
- 支持租户级资源配额控制（存储容量、并发任务数、报表数量）
- 支持跨租户管理员视图，平台管理员可查看所有租户数据
- 支持租户级个性化配置（水印开关、导出格式限制）

### 3.6 审计与合规需求

- 全量操作审计日志，记录所有用户行为（登录、预览、下载、导出、修改报表）
- 敏感操作审计（DDL变更、数据源配置修改、权限变更），支持审批流程
- 审计日志不可删除、不可篡改，支持按条件查询与导出
- 满足等保三级合规要求

### 3.7 可观测性需求

- 核心指标监控：导入/导出/PDF生成成功率、耗时、错误率
- 全链路追踪，支持从前端请求到后端任务的完整链路排查
- 结构化日志收集，支持日志聚合查询与告警配置
- 系统健康度大盘，实时展示各服务节点状态与资源使用率

## 4. 扩展功能需求

### 4.1 报表订阅与主动推送

- 支持定时订阅配置：每日/每周/每月自动生成报表并邮件推送
- 支持 Webhook 推送至企业微信/钉钉/Slack
- 异常数据告警：关键指标环比波动超过阈值自动通知
- 支持订阅权限控制，仅报表所有者可设置订阅规则

### 4.2 数据安全与防泄露

- 预览页动态添加用户水印（用户名+时间戳+IP），支持全屏浮水印
- 导出 PDF 自动添加隐形数字水印，可追溯泄露来源
- 支持分级权限控制：禁止下载/仅允许在线查看/允许导出
- 支持数据脱敏：敏感字段（手机号、身份证、金额）自动脱敏展示

### 4.3 文件安全扫描

- 上传文件自动病毒扫描，集成开源杀毒引擎
- 可疑文件自动移入隔离区，管理员审核后放行
- 支持文件类型深度检测，防止恶意文件伪装

### 4.4 报表模板市场

- 内置常用行业报表模板（财务、销售、人力、运营）
- 用户可将已配置报表发布为公共模板
- 模板版本管理，支持模板升级与回滚
- 模板评分与收藏功能

### 4.5 国际化支持

- 前端界面中英文切换
- 日期、数字、货币格式本地化适配
- 多语言报表内容支持（同一报表多语言版本）

### 4.6 性能基准

- 单文件最大支持 500MB 导入，100万行数据导入耗时 ≤ 5分钟
- 百页级复杂报表 PDF 生成耗时 ≤ 30秒
- 并发 50 个导出任务时，系统可用性 ≥ 99.5%
