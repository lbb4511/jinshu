## 1. 概述

### 1.1 合规目标

锦书企业级报表系统须满足**网络安全等级保护第三级（等保三级）**要求，确保系统在以下维度达到安全基线：

- 物理环境安全
- 通信网络安全
- 区域边界安全
- 计算环境安全
- 安全管理中心
- 安全管理体系

### 1.2 适用标准

| 标准编号        | 标准名称                                      |
| --------------- | --------------------------------------------- |
| GB/T 22239-2019 | 信息安全技术 网络安全等级保护基本要求         |
| GB/T 28448-2019 | 信息安全技术 网络安全等级保护测评要求         |
| GB/T 25070-2019 | 信息安全技术 网络安全等级保护安全设计技术要求 |

### 1.3 对标说明

| 符号 | 含义                                 |
| ---- | ------------------------------------ |
| ✅   | 已实现或已有对应方案                 |
| 🔶   | 部分实现，存在差距                   |
| ❌   | 未实现，需新建                       |
| ⚪   | 不适用（本项目范围外或 IaaS 层覆盖） |

## 2. 安全物理环境

> 本项目部署于 K3s 集群，底层物理环境由云平台/IaaS 提供商负责，本条不适用。

| 控制点     | 要求                    | 状态 | 说明               |
| ---------- | ----------------------- | ---- | ------------------ |
| 物理访问   | 机房出入管控、视频监控  | ⚪   | 云平台/IaaS 层覆盖 |
| 温湿度控制 | 机房温湿度自动调控      | ⚪   | 云平台/IaaS 层覆盖 |
| 电力供应   | 双路供电 + UPS + 发电机 | ⚪   | 云平台/IaaS 层覆盖 |
| 防火防水   | 自动消防 + 漏水检测     | ⚪   | 云平台/IaaS 层覆盖 |
