## 1. 测试策略总览

### 1.1 测试金字塔

```
         ╱╲
        ╱  ╲          E2E 测试（5%）
       ╱    ╲         Playwright, 关键业务流程
      ╱──────╲
     ╱        ╲      集成测试（25%）
    ╱          ╲     Testcontainers, API MockMvc
   ╱────────────╲
  ╱              ╲   单元测试（70%）
 ╱                ╲  JUnit 5 + Mockito, Vitest
╱──────────────────╲
```

### 1.2 测试优先级定义

| 等级 | 标签 | 说明 | CI 触发 |
|------|------|------|---------|
| P0 | `@Tag("P0")` | 核心功能必过，阻塞发版 | 每次 PR |
| P1 | `@Tag("P1")` | 重要功能，失败告警不阻塞 | 每次 PR |
| P2 | `@Tag("P2")` | 边界/异常/兼容性，定期回归 | Nightly |

### 1.3 执行环境

| 测试类型 | 环境 | 工具 | 数据源 |
|----------|------|------|--------|
| 单元测试 | 本地/CI | JUnit 5 + Mockito | 内存/Mock |
| DAO 集成测试 | Testcontainers PG | `@MybatisTest` | PostgreSQL 16 |
| Service 集成测试 | Testcontainers PG + Redis + MQ | `@SpringBootTest` | 容器化中间件 |
| API 集成测试 | Testcontainers + MockMvc | `@AutoConfigureMockMvc` | 容器化中间件 |
| E2E 测试 | 预发布环境 | Playwright / REST Client | 预发布数据库 |
