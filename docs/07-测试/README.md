## 测试金字塔

```
        / \
       / E \      E2E 端到端测试（少量核心流程）
      /-----\
     / 集成  \    集成测试（组件协同）
    /---------\
   /  单元测试  \  单元测试（大量，全覆盖）
  /-------------\
```

### 测试分布建议

| 测试类型 | 数量占比 | 执行速度 | 运行频率 |
| -------- | -------- | -------- | -------- |
| 单元测试 | 70%      | < 100ms  | 每次提交 |
| 集成测试 | 25%      | 几秒     | 每次 PR  |
| E2E 测试 | 5%       | 几分钟   | 每日构建 |
