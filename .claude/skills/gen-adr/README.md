# gen-adr 技能使用指南

自动生成符合项目标准的架构决策记录(ADR)文档。

## 用法

```bash
/gen-adr "决策标题" [编号]
```

## 示例

```bash
# 创建新的ADR
/gen-adr "使用行级多租户方案而非Schema隔离"

# 指定编号创建
/gen-adr "使用EasyExcel而非POI做Excel导入" --number 008

# 查看所有ADR列表
/gen-adr --list

# 只显示模板
/gen-adr --template
```

## ADR标准模板

每个ADR包含以下必填章节：

| 项 | 内容 |
|----|------|
| 编号 | ADR-XXX 连续编号 |
| 标题 | 一句话概括决策 |
| 创建日期 | YYYY-MM-DD |
| 状态 | 提议/已通过/已废弃/已替代 |

**正文内容：**
1. **背景与问题描述** - 为什么需要做这个决策
2. **考虑过的可选方案** - 至少列出2-3个方案
3. **最终决策** - 选了哪个方案
4. **决策理由** - 为什么选这个（利弊分析）
5. **已知缺点与风险** - 有什么trade-off
6. **后续影响** - 对架构、代码、团队的影响
7. **参考资料**（可选）
