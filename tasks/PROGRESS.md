## 🚨 关键缺口

基于最新审计，以下能力尚未启动或仅停留在字段占位：

1. **报表模板设计** — 仅有 `template_config` 字符串字段，无模板表/模板市场/Excel 模板导出 API
2. **RBAC 权限模型** — 仅有 `users.role` 字符串硬编码，无 role/permission/关联表
3. **画面一览** — 无独立页面清单文档，前端管理页（租户/用户）和报表编辑页缺失
4. **数据库迁移工具** — 仍用原始 DDL 脚本，无 Flyway/Liquibase
5. **敏感数据保护** — 数据源密码未加密，无脱敏/水印实现
6. **可观测性** — 监控、日志、告警、备份、灾备文档和配置均未完成
